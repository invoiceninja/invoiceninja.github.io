A quote (estimate) is a proposed set of charges you send to a client for
approval. The Quotes API lets you create, send, and manage quotes, and
convert an approved quote directly into an invoice.

### Common uses
- Send estimates for client approval via the client portal
- Convert an accepted quote into an invoice
- Track quote status (draft, sent, approved)
