A payment records money received from a client, optionally allocated
across one or more invoices. The Payments API lets you record manual
payments, apply credits, process refunds, and reconcile amounts so client
balances stay accurate.

### Common uses
- Record a payment and allocate it across one or more invoices
- Apply credits and handle partial or over-payments
- Refund or delete payments and adjust balances
